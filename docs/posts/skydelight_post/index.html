SkyDelight.

SkyDelight es una aplicación móvil Android para realizar “test” y recomendaciones relacionadas con el estrés en estudiantes universitarios de entre 18 a 25 años, la cuál tomé el rol el rol de diseñador UX y codificador.

Esto lo encontramos a través de una encuesta realizada por Microsoft Forms, se identificó que los estudiantes universitarios de ESCOM sufren de altos niveles de estrés. Por lo que les pareció interesante usar una aplicación que les de recomendaciones que los apoyé en su vida cotidiana.
